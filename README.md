# An-lisis-de-Ventas-y-Optimizaci-n-con-Herramientas-Avanzadas-en-Excel
Proyecto de análisis y optimización de ventas en Excel, enfocado en la limpieza, transformación y exploración de grandes volúmenes de datos mediante Power Query y tablas dinámicas avanzadas, facilitando la generación de insights para la toma de decisiones comerciales.
